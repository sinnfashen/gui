安装说明：
	解压后将之前的newspaper文件夹（即目前已经整理好的）放入本目录下（替换原来的空newspaper文件夹）即可使用。

使用方法：
	双击Stargazer.exe即可打开搜索页面。
	每栏的第一个是第一关键词，如果需要第二关键词，需在中间选择逻辑符号（AND OR），并在最后一个空填入第二关键词。
	搜索成功后会弹出提示框，部分关键词结果数量较大可能需要等待。
	搜索结果双击可以打开文件。
	
导入说明：
	点击导入后，选择要导入的newspaper文件夹即可。
	注意1：文件夹的层次要求见下。
	注意2：导入的过程中会将文件复制到本目录下的newspaper中对应位置，数量大时可能耗时较长，请耐心等待。

文件结构:
	顶层命名newspaper
	第一层为1,2,3这样的报纸名字
	第二层001000这样的股票名字
	第三层为txt文件
